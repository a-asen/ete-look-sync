# etesync-outlook-calendar-sync

Headless mirror of a UiT Outlook calendar to a personal EteSync calendar.

Node/TypeScript rewrite of
[`etesync-outlook-calendar-scraper-sync`](https://github.com/a-asen/etesync-outlook-calendar-scraper-sync)
— same goal, but built around the actively-maintained `etebase` npm SDK
instead of the abandoned Python one.

> **Status: not usable yet.** Active rewrite in progress. See
> [`PLAN.md`](PLAN.md) for the roadmap and current phase.
> The Python project remains the working tool until cutover.
